# 快速开始

> English summary: This guide explains how to start Engineering Photo Map locally with Docker Compose. All values are examples and should be adjusted for your environment.

本文说明如何在本机使用 Docker Compose 启动工程照片地图管理系统。所有示例值都可以按你的环境调整。

## 准备条件

- Docker Desktop 或 Docker Engine
- Docker Compose
- Node.js 只在本地单独开发前端时需要；使用 Docker Compose 时会在容器内安装依赖

## 启动步骤

1. 复制环境变量模板：

```powershell
Copy-Item .env.example .env
```

2. 修改 `.env` 中的数据库密码：

```text
POSTGRES_PASSWORD=change_this_password
```

请把示例值替换为你自己的强密码，不要在公开仓库提交真实 `.env`。

3. 启动服务：

```powershell
docker compose up --build
```

4. 打开前端页面：

```text
http://localhost:<FRONTEND_PORT>
```

5. 检查 API：

```text
http://localhost:<API_PORT>/api/health
```

`FRONTEND_PORT` 和 `API_PORT` 来自 `.env`。公开文档只使用占位符，不记录真实部署端口。

## 停止服务

```powershell
docker compose down
```

如果需要保留数据，不要删除 `HOST_DATA_DIR` 指向的目录。
