# 快速开始

本文说明如何在本机用 Docker Compose 启动 Engineering Photo Map。所有示例值都可以按你的环境调整。

## 准备条件

- Docker Desktop 或 Docker Engine
- Docker Compose
- Node 只在本地单独开发前端时需要；使用 Docker Compose 时会在容器内安装依赖

## 启动步骤

1. 复制环境变量模板：

```powershell
Copy-Item .env.example .env
```

2. 修改 `.env` 中的数据库密码：

```text
POSTGRES_PASSWORD=change_this_password
```

3. 启动服务：

```powershell
docker compose up --build
```

4. 打开页面：

```text
http://localhost:5173
```

5. 检查 API：

```text
http://localhost:8000/api/health
```

## 停止服务

```powershell
docker compose down
```

如果需要保留数据，不要删除 `HOST_DATA_DIR` 指向的目录。
