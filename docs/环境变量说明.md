# 环境变量说明

> English summary: This document lists the public environment variables used by Engineering Photo Map. Example values are safe placeholders; replace secrets and deployment-specific values privately.

本文说明工程照片地图管理系统使用的公开环境变量。示例值仅用于说明，真实密码、域名、地址、端口和路径请保存在私有 `.env` 中。

## 数据库与 Redis

| 变量 | 说明 | 示例 |
| --- | --- | --- |
| `POSTGRES_DB` | PostgreSQL 数据库名 | `photo_map` |
| `POSTGRES_USER` | PostgreSQL 用户名 | `photo_user` |
| `POSTGRES_PASSWORD` | PostgreSQL 密码，生产环境必须修改 | `change_this_password` |
| `REDIS_URL` | Redis 连接地址 | `redis://redis:6379/0` |

## 服务端口

| 变量 | 说明 | 示例 |
| --- | --- | --- |
| `API_PORT` | API 暴露到宿主机的端口 | `<API_PORT>` |
| `FRONTEND_PORT` | 前端暴露到宿主机的端口 | `<FRONTEND_PORT>` |
| `POSTGRES_PORT` | PostgreSQL 暴露端口，本地开发可用 | `<POSTGRES_PORT>` |
| `REDIS_PORT` | Redis 暴露端口，本地开发可用 | `<REDIS_PORT>` |

公开文档使用占位符描述端口。请在私有 `.env` 中填写真实部署值。

## 数据目录

| 变量 | 说明 |
| --- | --- |
| `HOST_DATA_DIR` | 宿主机持久化数据目录 |
| `DATA_DIR` | 容器内部数据根目录 |
| `ORIGINALS_DIR` | 原图目录 |
| `UPLOADS_DIR` | 上传暂存目录 |
| `THUMBNAILS_DIR` | 缩略图目录 |
| `PREVIEWS_DIR` | 预览图目录 |
| `EXPORTS_DIR` | 导出目录 |
| `TEMP_DIR` | 临时目录 |
| `LOG_DIR` | 日志目录 |
| `REPORTS_DIR` | 报告目录 |
| `BACKUPS_DIR` | 备份目录 |

`HOST_DATA_DIR` 可以是 `./data`、NAS 上的持久化目录或其他受保护的路径。公开仓库不应包含这些运行数据。

## 访问与跨域

| 变量 | 说明 | 示例 |
| --- | --- | --- |
| `PUBLIC_BASE_URL` | 前端公开访问地址 | `https://photos.example.com` |
| `API_BASE_URL` | API 公开访问地址 | `https://api.photos.example.com` |
| `VITE_API_BASE_URL` | 前端构建或运行时 API 地址 | `https://api.photos.example.com` |
| `CORS_ORIGINS` | 允许访问 API 的前端来源 | `https://photos.example.com,http://localhost:<FRONTEND_PORT>` |
| `VITE_ALLOWED_HOSTS` | Vite 允许的请求 Host | `photos.example.com,api.photos.example.com` |

`example.com` 和 `<...>` 是公开文档占位符，不代表真实部署地址。

## 上传限制

| 变量 | 说明 | 示例 |
| --- | --- | --- |
| `MAX_UPLOAD_MB` | 单文件上传大小限制 | `50` |
| `ALLOWED_IMAGE_EXTENSIONS` | 允许上传的图片扩展名 | `jpg,jpeg,png,heic,webp,tif,tiff` |
| `MAX_UPLOAD_FILES` | 批量上传文件数量限制 | `999` |
| `VITE_MAX_UPLOAD_FILES` | 前端批量上传数量限制 | `999` |
| `VITE_UPLOAD_CONCURRENCY` | 前端上传并发数 | `3` |
