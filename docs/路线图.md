# 路线图

> English summary: This roadmap describes planned improvements after the first public release. It is indicative and may change with community feedback.

本文记录首次公开发布后的计划方向。路线图不是承诺清单，会根据实际使用反馈调整。

## 近期

- 完善公开部署文档。
- 增加更稳定的生产前端构建方案。
- 增加更多后端和前端自动化测试。
- 优化批量上传和 Worker 状态展示。

## 中期

- 增强 QGIS、CSV、GeoJSON 导出能力。
- 增加更完善的项目归档和备份入口。
- 增强地图打印和照片清单输出。
- 改进权限控制和多用户协作能力。

## 长期

- 支持更完整的生产部署模板。
- 增加对象存储或远程备份适配。
- 增加更细粒度的安全审计和数据脱敏工具。
